Welcome to the libnrm guide. This document will help you in your use of the
C/C++ and Fortran interface for NRM.

Install
=======

The libnrm code can be installed from source::

 git clone https://github.com/anlsys/libnrm.git
 cd libnrm
 ./autogen.sh
 ./configure
 make && make install
